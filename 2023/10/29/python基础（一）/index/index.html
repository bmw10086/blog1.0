python基础（二）

一、注释

作用：使用自己熟悉的语言，在程序中对某些代码进行标注说明，增强程序的可读性。

1.单行注释

#（可空格）注释内容 注：为了代码的可读性建议#后加个空格

在代码后面增加单行注释 在代码与注释之间至少有两个空格

2.多行注释

‘’‘’‘’

‘’‘’‘’

3、中文编码声明注释

二、保留字（关键字）与标识符

保留字是python语言中已经被赋予特定意义的一些单词，所有保留字都是区分字母大小写的

可以通过命令查看关键字： import关键字可以导入一个工具包

标识符可以理解一个名字，它主要用来标识变量、函数、类、模块和其他对象名称（起名字要见名知义）

  • 由字母、下划线和数字组成，首字母不能是数字
  • 不能使用python保留字
  • 区分大小写

变量的命名规范：

  • 见名知意
  • 下划线命名法
  • 英文字母全小写

三、变量

程序就是用来处理数据的，而变量就是用来存储数据的

变量名：通常用字母、数字、和下划线构成，不能以数字开头，并且区分打大小写

python3支持中文，也可以做为变量名

变量的格式：变量名 = 变量值

1、变量的定义与命名

  • 在python中每个变量在使用前必须赋值，变量赋值以后该变量才会被创建
  • 等号（=）用来给变量赋值

变量的命名规则

命名规则可以被视为一种惯例，并无绝对与强制，目的是为了增加代码的识别和可读性。

  1. 在定义变量时，为了保证代码格式，“**=**”的左右应该各保留一个空格

  2. 在Python中，如果变量名需要由二个或多个单词组成时，可以按照以下方式命名

  • 每个单词都使用小写字母，
  • 单词与单词之间使用**_**下划线连接
  • 例如：first_name、last_name、qq_number、qq_password

驼峰命名法

当变量名是由二个或多个单词组成时，还可以利用驼峰命名法来命名

小驼峰式命名法

第一个单词以小写字母开始，后续单词的首字母大写

例如：firstName、lastlame

大驼峰式命名法

每一个单词的首字母都采用大写字母

例如：FirstName、LastName、

2、变量的类型

四种：

  • str 字符串
  • bool 布尔（真假）
  • int 整型
  • float 浮点数（小数）

注：python中定义变量时不需指定类型，python可以根据=等号右侧的值，自动推导出变量中存储数据类型。（其他语言需要）

使用内置函数type()可以返回变量类型

%被称为格式化操作符，专门用于处理字符串的格式。

  • 包含%的字符串，被称为格式化字符串
  • %和不同的字符连用，不同类型的数据需要使用不同的格式化字符

四、基本数据类型

数据类型可以分为数字型和非数字型

1、数字类型

  • 整数

  • 浮点数

    精确计算浮点数：decimal模块

  • 复数

    由实部和虚部组成，并且使用J和j表示虚部

2、字符串类型

3、布尔类型

布尔类型主要来表示真或假的值，在python中True 和 False被解释为布尔值

在python中所有对象都可以进行真值测试，只有以下几种得到的值为假：

  • False或None
  • 数值中的零。包括0、0.0、虚数0
  • 空序列，包括字符串、空元组、空列表、空字典
  • 自定义对象实例，对象的____bool____方法返回False或者_len_方法返回0

变量是bool型，在计算时

True对应的数字是1

False对应的数字是0

使用bool（）内置函数直接给出True和False

使用type () 查看数据的类型，并且返回结果

变量无类型，但是它存储的数据有

4、数据类型的转换

语句（函数） 说明
int(x) 将x转换为整数
float(x) 将x转换为浮点数
str(x) 将对象x转换为字符串
  • 任何类型都可转换为字符串
  • 字符串不可以随意转换为数字，必须为数字
  • 浮点数转换为整数会丢失精度

五、运算符

1、算数运算符

在Pytho中 * 运算符还可以用于字符串，计算结果就是字符串重复指定次数的结果

如：”hello” * 100

优先级：

先乘除后加减

同级运算符是从左至右计算

可以使用（）调整计算的优先级

2、赋值运算符

在python中，使用 = 可以给变量赋值

注：赋值运算符中间不能使用空格

3、比较（关系）运算符

通过比较的运算符得到的都是布尔类型

4、逻辑运算符

一、and

语法：

条件一 and 条件二

  • 与 并且
  • 两个条件同时满足，返回True
  • 只要有一个不满足，就返回Flase

二、or

语法：

条件一 or 条件二

  • 或 或者
  • 两个条件只要有一个满足，返回True
  • 两个条件都不满足，返回Flase

三、not

语法：

Not 条件

  • 非 不是

在Python中，要使用随机数，首先需要导入随机数的模块–”工具包”

Import random

导入模块后，可以直接在模块名称后面敲一个。然后按Tab键，会提示该模块中包含的所有函数
